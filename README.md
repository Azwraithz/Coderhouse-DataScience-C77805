# 📊 Clasificación de Tickets de Soporte con Deep Learning

## Descripción del Proyecto
Este proyecto desarrolla un pipeline completo de Procesamiento de Lenguaje Natural (NLP) para clasificar tickets de soporte al cliente. El objetivo es optimizar los tiempos de respuesta mediante la categorización automática de problemas.

## Stack Tecnológico
* **Lenguaje:** Python 3.x
* **NLP:** NLTK & spaCy
* **Vectorización:** TF-IDF (Scikit-Learn)
* **Modelado:** PyTorch (Redes Neuronales Densas)
* **Visualización:** Matplotlib & Seaborn

## Estructura del Proyecto
* `data/`: Datasets originales y procesados.
* `notebooks/`: Notebook principal con la Evaluación Final.
* `README.md`: Documentación del proyecto.
